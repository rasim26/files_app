## 🚀 Features

- ✅ Create new items
- ✅ View list of all items
- ✅ Edit existing items
- ✅ Delete items with confirmation

---

## 🛠️ Tech Stack & Concepts Applied

| Concept | Usage |
|--------|-------|
| `StatefulWidget` | Managing local UI state |
| `Firestore` | Storage |
| `ListView.builder` | Rendering dynamic lists |
| `Navigator` | Screen routing |
| `Image picker` | image Input |

- Building reactive UIs with Flutter widgets
